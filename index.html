자료 정리, 수치 검토, 신고 업무 전반을 데이터로 체계화하는 회계·세무 담당자

회계학과·빅데이터 다전공 기반의 데이터 역량을 회계·사무 실무에 적용

소개

학력 한남대학교 회계학과·빅데이터응용학과 다전공 4학년 재학 중 (2027년 2월 졸업 예정)
소개 세무법인 인턴으로 자료 정리, 수치 검토, 전자신고 보조 등 회계·사무 실무를 경험했습니다. 회계학과·빅데이터 다전공과 민관데이터 프로젝트 경험을 바탕으로, 데이터를 활용한 체계적 업무 수행 역량을 갖추고 있습니다.

좌우명

실패가 지닌 가치를 인정한다. 나는 실패를 그냥 털어버리지 않고 실패가 지닌 중요한 가치를 추출해낸다.

강점

  • 업무를 미루지 않고 제때 처리하는 신속한 실행력
  • 체계적인 성격과 일관된 업무 처리 방식
  • 회계학적 지식과 데이터 처리 능력
  • 피드백 수용력이 높고, 실패를 학습으로 전환하는 태도
  • 한 번 맡은 일은 끝까지 완수하는 책임감

How I Work

저는 일을 감으로 처리하기보다, 구조화된 사고를 통해 해결합니다.

문제를 구조로 나누는 편

감으로 처리하기보다
문제 정의 → 선택지 → 판단 기준 → 결정
단계로 정리한 후 움직입니다.

손이 빠르고 미루지 않음

업무를 작은 단위로 쪼개 즉시 실행합니다.
일정 지연을 막기 위해 선처리–후보완 방식을 선호합니다.

실패를 데이터로 남김

결과가 기대와 달라도 좌절로 끝내지 않습니다.
원인을 정리하고 기준을 업데이트합니다.
